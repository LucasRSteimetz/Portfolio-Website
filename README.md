# Portfolio Website

Simple website created using HTML and CSS. I still intend to develop its backend in the future.

## How to run it

Firt, git clone this repository:

```
git clone https://github.com/LucasRSteimetz/Portfolio-Website.git
```

To run it, you have two options:

### First Option: 

Open *index.html* directly on browser.

### Second Option

Run the following command to run a local server:

```
python3 -m http.server 8000
```

And then search for *http://localhost:8000* in your browser's search bar.

Note: You need to have Python version 3.12 or a more recent version installed.
